# Creating Matter Applications using SLC CLI

**Silicon Labs Configurator (SLC)**: SLC offers command-line access to application configuration and generation functions. [Overview of the SLC-CLI](https://docs.silabs.com/command-line-development/latest/ssv6-slc-cli/) provides instructions on downloading and using the SLC-CLI tool.

Instructions on how to build Matter projects using SLC CLI can be found in the README section of [Matter Extension Repo](https://github.com/SiliconLabsSoftware/matter_extension).
